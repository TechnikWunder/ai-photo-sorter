# ai-photo-sorter
a python script to sort your photos


dependencies: ollama, python

to install the python packages on Windows run: pip install -r requirements.txt

to install the python packages on MacOS or Linux run: pip3 install -r requirements.txt


to start put your photos in /photos_input supported are jpg and heic (heic will automaticly converted in jpg) when you have a live photo from an apple device put the jpg/heic and the mov in /photos_input and the mov will moved to the jpg or heic when it has the same name
